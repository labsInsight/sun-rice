Introduction

sun-rice

Mobile UI elements for Vue 2.0

Demo

  • demo 地址
  • document 地址

Installation

npm i sun-rice -S

Usage

Import all components.

import Vue from 'vue'
import { install as SunRice } from 'sun-rice'
import 'sun-rice/dist/static/css/sun-rice.css'

Vue.use(SunRice)

Or import specified component. (Use babel-plugin-component)

import { SunCell } from 'sun-rice'

Vue.component(SunCell.name, SunCell)

babel-plugin-component

  • Auto import css file
  • Modular import component

Installation

npm i babel-plugin-component -D

Usage

.babelrc

{
  "plugins": [
    "other-plugin",
    ["component", [{ "libraryName": "sun-rice", "style": true }]]
  ]
}

Development

npm run dev
